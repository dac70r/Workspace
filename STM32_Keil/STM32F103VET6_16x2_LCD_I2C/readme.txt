This project features a 16x2 LCD interfaced with STM32F103VET6 based microcontroller using the I2C protocol.

Microcontroller is capable of printing text on the 16x2 LCD display. Also features a clock (inaccurate). 

LCD Library obtained from: https://embeddedthere.com/interfacing-stm32-with-i2c-lcd-with-hal-code-example/
